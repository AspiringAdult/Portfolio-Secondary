# KINGSLAYER Portfolio

Cinematic Next.js portfolio for Diptangkush Das, built with Next.js, Three.js, Framer Motion, and Tailwind CSS.

## Local Dev

```bash
yarn install
yarn dev
```

## Deployment

Connect the GitHub repo to Vercel, set the root directory to this app folder, and deploy. No env vars required.

The build command is `next build`, and the output directory is `.next`.

## Assets

`public/Diptangkush_CV.pdf` is included for the resume link.

Add `public/og-image.png` before launch for rich social previews.
